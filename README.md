Browser History Manager Data Structures Assignment 02This repository contains a C++ implementation of a core data engine for a lightweight web browser. Built entirely using raw pointers and custom classes without any STL containers, this project demonstrates real-world applications of Linked Lists.🚀 Features1. Bookmarks Manager (Singly Linked List) Implements a Singly Linked List (SLL) to store saved websites.Automatically maintains bookmarks in alphabetical order upon insertion.Allows users to pin a favorite bookmark directly to the top of the list.Includes functionality to search, remove specific URLs, and count total saved bookmarks.Features a clearAll() method to delete every node and reset the list.2. Tab History Navigator (Doubly Linked List) Implements a Doubly Linked List (DLL) to track every page visited in the current session.Supports real browser-like Back and Forward navigation.Automatically erases forward history if a user goes back and visits a new page.Simulates an incognito mode that instantly clears all browsing history and resets pointers.Includes an in-place history reversal feature to view pages from oldest to newest.🛠️ Tech StackLanguage: C++ Core Concepts: Singly Linked Lists, Doubly Linked Lists, Memory Management, Custom Node Classes.
